## Installation

Es gibt jede Menge Auswahl an Tools die einen Kubernetes auf seinen eigenen Rechner installieren.
Auf der Suche findet man Tools wie k8s, k3s, k0s, k3d, minikube usw.

Auf macOS kann ich aber [Orbstack](https://orbstack.dev/) empfehlen, welches mit [Kubernetes-Unterstützung](https://docs.orbstack.dev/kubernetes/) daher kommt.

Der große Vorteil von Orbstack ggü. den o.g. Tools ist,
dass die aufgesetzten Container direkt (ohne zusätzliches Hinzufügen in die `/etc/host`)
unter `<service-name>.<namespace>.svc.cluster.local` zu erreichen sind.
Das erleichtert das Aufsetzen, Testen, Debuggen und die Nutzung enorm. Dazu später mehr.

Nach der Installation von Orbstack, muss Kubernetes erst noch gestartet werden:

```sh
orb start k8s
```

> `orb` ist das CLI-Tool von orbstack.
> Falls `orb` nicht gefunden wurde, gehe auf https://docs.orbstack.dev/install und stelle sicher, dass die Installation erfolgreich war.


Stelle sicher dass folgender Befehl funktioniert:

```sh
> kubectl config get-contexts
CURRENT   NAME       CLUSTER    AUTHINFO   NAMESPACE
*         orbstack   orbstack   orbstack
```
